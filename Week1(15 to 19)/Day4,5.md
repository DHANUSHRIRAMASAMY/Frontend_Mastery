Git Workflow & Basic Commands
git init – Initializes a new Git repository in the project folder
git clone – Copies an existing repository from GitHub to local system
git status – Shows the current status of changes in the repository
git add . – Adds all changes to the staging area
git commit -m "message" – Saves changes with a commit message
git push – Uploads local changes to the remote repository (GitHub)
git pull – Fetches and merges latest changes from remote repository
git branch – Shows or creates branches in the repository
git checkout – Switches between branches
git merge – Combines changes from different branches
