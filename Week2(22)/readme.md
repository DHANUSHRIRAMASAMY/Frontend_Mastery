JavaScript Basics
Variables
The three ways of declaring variables:
let – used for values that can be changed later.
const – used for values that should remain constant and cannot be reassigned.
var – the older way of declaring variables, which is function-scoped.


Data Types
Primitive Data Types
String – Stores text values.
Number – Stores integer and decimal numbers.
Boolean – Stores true or false.
Undefined – A variable declared without a value.
Null – Represents an intentional empty value.
BigInt – Used for very large integers.
Symbol – Used to create unique identifiers.

Non-Primitive Data Types
Object – Stores data in key-value pairs.
Array – Stores multiple values in a single variable.
Function – A reusable block of code that performs a specific task.


Expressions
Learned that an expression is any piece of code that produces a value.
Explored different types of expressions such as arithmetic, assignment, comparison, logical, and function call expressions.


Operators
Studied various operators used to perform operations on values and variables:
Arithmetic Operators (+, -, *, /, %)
Assignment Operators (=, +=, -=, etc.)
Comparison Operators (==, ===, >, <, >=, <=)
Logical Operators (&&, ||, !)
Increment and Decrement Operators (++, --)
