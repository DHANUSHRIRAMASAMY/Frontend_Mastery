# JavaScript Functions and Parameters
# Function
A function is a reusable block of code that performs a specific task.
# Function Declaration
A named function created using the function keyword.
# Function Expression
A function stored inside a variable.
# Arrow Function
A shorter way to write a function using the => operator.
# Anonymous Function
A function that does not have a name.
# Callback Function
A function passed as an argument to another function.
# Recursive Function
A function that calls itself until a condition is met.
# Higher-Order Function
A function that accepts another function as an argument or returns a function.
# Constructor Function
A special function used to create objects with similar properties and methods.
# IIFE (Immediately Invoked Function Expression)
A function that runs immediately after it is created.
# Generator Function
A special function that can pause and resume its execution using yield.
# Async Function
A function used to handle asynchronous operations such as API calls and data fetching.

# Parameters
Parameters are variables defined in the function declaration that receive values.
# Arguments
Arguments are the actual values passed to a function when it is called.
# Return Statement
The return statement sends a value back from a function.

# Types of Expressions

# Arithmetic Expression
Used to perform mathematical calculations.
Examples include addition, subtraction, multiplication, and division.
# Assignment Expression
Used to assign a value to a variable.
Helps store data for later use.
# Comparison Expression
Used to compare two values.
Produces either true or false.
# Logical Expression
Used to combine conditions.
Helps make decisions in programs.
# String Expression
Used to work with text values.
Can combine multiple strings into one.
# Conditional (Ternary) Expression
A short way to write simple if-else conditions.
Returns one value if the condition is true and another value if it is false.
# Function Call Expression
Created when a function is called.
Produces the value returned by the function.
# Object Expression
Used to create an object.
Stores data as key-value pairs.
# Array Expression
Used to create an array.
Stores multiple values in a single variable.
# Regular Function Expression
A normal function stored in a variable.
Can be called using the variable name.
# Generator Function Expression
A special function that can pause and continue its execution.
Useful when values need to be generated one at a time.
# Class Expression
A class stored in a variable.
Used to create objects and organize code.
