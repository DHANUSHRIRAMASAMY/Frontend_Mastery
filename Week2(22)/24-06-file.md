# Day 24-06-26- Arrays & Objects in JavaScript

## Arrays

### Array Declaration
```javascript
let fruits = ["Apple", "Banana", "Mango"];
```

### Accessing Elements
```javascript
console.log(fruits[0]);
```

### Updating Elements
```javascript
fruits[1] = "Orange";
```

### Array Length
```javascript
console.log(fruits.length);
```

## Array Methods

### push()
```javascript
fruits.push("Grapes");
```

### pop()
```javascript
fruits.pop();
```

### unshift()
```javascript
fruits.unshift("Kiwi");
```

### shift()
```javascript
fruits.shift();
```

### splice()
```javascript
fruits.splice(1, 1, "Pineapple");
```

### slice()
```javascript
let newFruits = fruits.slice(0, 2);
```

### concat()
```javascript
let allFruits = fruits.concat(["Guava", "Papaya"]);
```

### join()
```javascript
console.log(fruits.join(", "));
```

### includes()
```javascript
console.log(fruits.includes("Mango"));
```

### indexOf()
```javascript
console.log(fruits.indexOf("Mango"));
```

### forEach()
```javascript
fruits.forEach(fruit => console.log(fruit));
```

### map()
```javascript
let upper = fruits.map(fruit => fruit.toUpperCase());
```

### filter()
```javascript
let result = fruits.filter(fruit => fruit.length > 5);
```

### reduce()
```javascript
let sum = [1, 2, 3, 4].reduce((a, b) => a + b, 0);
```

### sort()
```javascript
fruits.sort();
```

### reverse()
```javascript
fruits.reverse();
```

## Objects

### Object Declaration
```javascript
let student = {
  name: "John",
  age: 20
};
```

### Accessing Properties
```javascript
console.log(student.name);
console.log(student["age"]);
```

### Updating Properties
```javascript
student.age = 21;
```

### Adding Properties
```javascript
student.city = "Chennai";
```

### Deleting Properties
```javascript
delete student.city;
```

### Nested Objects
```javascript
let person = {
  name: "John",
  address: {
    city: "Chennai"
  }
};
```

## Object Methods

### Object.keys()
```javascript
console.log(Object.keys(student));
```

### Object.values()
```javascript
console.log(Object.values(student));
```

### Object.entries()
```javascript
console.log(Object.entries(student));
```

### hasOwnProperty()
```javascript
console.log(student.hasOwnProperty("name"));
```

### Object Destructuring
```javascript
let { name, age } = student;
```

### Spread Operator
```javascript
let updatedStudent = { ...student, grade: "A" };
```

slice() – Extract a portion of an array
concat() – Merge arrays
join() – Convert array to string
includes() – Check if an element exists
indexOf() – Find the index of an element
forEach() – Iterate through array elements
map() – Create a new transformed array
filter() – Filter elements based on a condition
reduce() – Reduce array to a single value
sort() – Sort array elements
reverse() – Reverse array elements
